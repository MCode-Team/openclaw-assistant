# Quickstart

## 1. Get Deepgram API Key

Visit https://console.deepgram.com/ to register and obtain an API Key (new users typically get $200 in free credits).

## 2. Configure API Key

Create a `.env` file:

```bash
DEEPGRAM_API_KEY=your_api_key
```

## 3. Start Application

```bash
npm start
```

## 4. How to Use

1. Click the microphone icon.
2. Speak: "What emails do I have today?"
3. View real-time recognition results and command execution.

For detailed documentation, please refer to `DEEPGRAM_SETUP.md`.
